📋 QUICK SETUP (10 minutes total)
---------------------------------

1. EXTRACT PROJECT:

2. INSTALL FRONTEND DEPENDENCIES (3 minutes):

   npm install
   # Wait for installation to complete

3. INSTALL BACKEND DEPENDENCIES (3 minutes):
   cd ../backend
   npm install
   # Installs Google Generative AI, PDF processing, OCR libraries

4. GET FREE GEMINI API KEY (2 minutes):
   # Visit: https://aistudio.google.com/app/apikey
   # Sign in with Google account
   # Click "Create API Key" - 100% FREE!
   # Copy key (starts with AIza...)

   # Edit backend/.env file:
   GEMINI_API_KEY=your_actual_gemini_key_here

5. RUN THE APPLICATION (2 minutes):
   # Terminal 1 - Backend:
   cd backend
   npm run dev

   # Terminal 2 - Frontend:
   cd frontend
   npm start
